# Chatbot RAG - Ambassade de Côte d'Ivoire à Paris

Ce projet est un chatbot RAG hautement sécurisé destiné à répondre aux questions des usagers de l'Ambassade.

## Installation

1. Cloner ou télécharger le dépôt.
2. Créer un environnement virtuel : `python -m venv .venv`
3. Activer l'environnement :
   - Windows : `.venv\Scripts\activate`
   - Linux/Mac : `source .venv/bin/activate`
4. Installer les dépendances : `pip install -r requirements.txt`
5. Copier `.env.example` vers `.env` et ajouter vos clés API.

## Lancement

Exécuter l'application Streamlit :
```bash
streamlit run streamlit_app.py
```
